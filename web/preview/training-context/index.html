Executive Summary

The Leverage Point

After 3 rounds of adversarial hardening that fixed 43 production bugs, a clear architectural weakness emerged: no centralized device/dtype context. Each component independently infers tensor placement, creating N independent decisions that must all agree.

Core insight: The elegant ambient context pattern (thread-local, invisible) is exactly what caused the bugs. Explicit parameter passing is less elegant but makes dependencies visible, errors obvious, and debugging tractable.
