Executive Summary

What This Changes

Core insight: SyncState already exists with all the right methods. The problem isn't design — it's wiring. This implementation connects the existing solution to the existing problem.

The 10-round adversarial hardening identified scattered mutable state as the root cause of 20+ bug classes. Five boolean flags across four modules, each with its own synchronization logic, each a potential race condition.

The fix: dependency injection of a single SyncState instance from Trainer into WeightBus, WeightLoader, and apply_lora_dropout. Plus a context manager to make cleanup impossible to forget.
